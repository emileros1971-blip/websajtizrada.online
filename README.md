# websajtizrada.online — production static website

Complete multilingual static website ready for GitHub Pages or standard hosting. No build process is required.

## Structure

- `index.html` and root `.html` files — Serbian version
- `en/` — complete English version
- `hu/` — complete Hungarian version
- `assets/css/style.css` — shared production stylesheet
- `assets/js/main.js` — shared behavior, contact configuration, consent-gated analytics and Meta Pixel
- `assets/images/portfolio/` — eight portfolio previews
- `.nojekyll`, `404.html`, `robots.txt`, `sitemap.xml` — GitHub Pages and SEO support

## Deployment

Upload the contents of this directory directly to the root of the GitHub repository or hosting account. Keep the folder structure unchanged. `index.html` must remain at the repository root.
